# Gestion Flextory

## Installation

### Base de donnée

Modifier le fichier d'environnement :
- APP_URL
- DB_HOST
- DB_PORT
- DB_DATABASE
- DB_USERNAME
- DB_PASSWORD

### Commandes

- `composer install`

- `npm ci`

- `php artisan migrate`

## Utilisation

Pour lancer le serveur Laravel : `php artisan serve`

Pour lancer la compilation une fois : `npm run build`

Pour lancer la compilation à chaque modification : `npm run watch`
